

Los modelos definen qué tipo de datos se esperan y cómo están organizados. 
En Angular, los modelos se utilizan para definir 
la estructura de los datos que se obtienen de una API.

Ejemplo: 
    - Si el backend nos envía un JSON con:

    {
        "nombre": "Messi",
        "email": "messi@gmail.com",
        "edad": 20,
        "club":{
            "club_nombre": "Inter miami",
            "club_pais": "estados unidos"
        }
    }

    el model será:
    export class User {
        nombre: string;
        email: string;
        edad: number;
        club: Club;
    }



- Acá van todas las clases que vamos a usar
- Ejemplo: Usuario, Vehiculo, etc.


Para crear una nueva clase:
1. Abrir terminal (en Visual Studio Code: CTRL + Ñ)
2. Escribir:
    ng generate class Models/nombreDeClase
        
    ejemplo: ng generate class Models/usuario
             ng generate class Models/vehiculo
