

 
- En los Services conectamos el Frontend con el Backend
- Sirven para enviar y recibir información desde el Backend

- Sirven para enviar toda la información a los Components que la necesiten


Para crear un nuevo Service:

1. Abrir terminal (en Visual Studio Code = CTRL + Ñ)
2. Escribir:
    ng generate service Services/nombreDelService
        
    ejemplo: ng generate service Services/usuario
             ng generate service Services/vehiculo
